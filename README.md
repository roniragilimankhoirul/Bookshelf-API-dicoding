# Bookshelf-API-dicoding

API untuk tugas akhir kelas backend-dasar Dicoding

## Usage

1. Clone the repository:

```shell
git clone https://github.com/roniragilimankhoirul/Bookshelf-API-dicoding.git
```

2. Install dependencies:

```shell
npm install
```

3. Start the server:

```shell
npm run start
```
